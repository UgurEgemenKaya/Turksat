Projeyi çalıştırmak için gerekli adımlar : 

1) Herhangi bir IDE'de proje açılır. (JAVA)
2) TestRunner Class'ı çalıştırılır. (sadece belge oluşturma senaryosu çalışacaktır.)


Proje detayları :

1) Belge oluşturma senaryosu otomasyonu

2) Bilgileri alan kontrolleri gherkin dili ile yazımı

3) Test raporları (pass/fail oranları, fail olduysa hangi adımda fail olduğunun ekran görüntüsünü ve daha bir çok istatistiği içerir.)

4) Proje JAVA 1.8, Selenium, TestNg, Cucumber, Maven  dil/framework/teknolojileri ile yazılmıştır. (version detaylarına POM.xml'den ulaşabilirsiniz.




Not: Test raporlarını elde edebilmek için, Projeyi açtığınız cihazda öncelikle Scoop ve Allure olmalıdır.("https://scoop.sh/" adresinden indirilebilir.)
("https://docs.qameta.io/allure/" adresinden indirilebilir.)

test sonucunu otomatik olarak sunan rapora ulaşılabilir.
